# 鱼书学习笔记

这是我在自学 PyTorch 和深度学习基础过程中整理的一套《**深度学习入门：基于 Python 的理论与实现**》学习笔记仓库。

这份仓库的目标不是简单摘抄原书内容，而是结合自己的理解，对书中的核心概念、公式、代码与章节逻辑进行重新梳理，整理成更适合复习与回顾的笔记版本。

## 笔记特点

相比直接阅读原书，我希望这份笔记尽量做到：

1. **结构清晰**  
   按章节整理，尽量保留原书主线逻辑。

2. **适合复习**  
   重点突出核心概念、关键公式和代码实现思路。

3. **兼顾原理与实现**  
   不只讲“是什么”，也关注“怎么写”“为什么这样写”。

4. **方便二次修改**  
   使用 Markdown / LaTeX 形式，方便后续继续补充和重排版。

## 学习路线说明

我目前的学习思路大致是：

1. 先通过鱼书打好神经网络与深度学习基础  
2. 边读边整理笔记，巩固概念和代码细节  
3. 再继续学习 PyTorch，逐步从“手写实现”过渡到实际框架开发  
4. 之后尝试做一些小项目，把理论和代码真正结合起来  

所以这个仓库本质上也是我整个深度学习入门过程的记录。
